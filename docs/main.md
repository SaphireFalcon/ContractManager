# Contract Manager

This documentation aims to give an overview of how to use this mod to create contract and missions.

## Contract and Blueprints

Contracts are offered from a blueprint if the requirements to offer the contract are met.
This contract instance is living within the game state and saved as part of the gamesave.

## Missions

Multiple contract blueprints can be grouped together to form a set of contracts that have be done.
Similar to contract blueprints these missions are specified as a blueprint.
